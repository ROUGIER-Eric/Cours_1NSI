Type construit¶

Un tableau index est un type construit car c'est un objet dont les lments sont des objets de types de base (ou ventuellement construit) :

Exemple

T = [4, 7, 8, 1]

Out[2]:
list
  • La variable T est un objet de type list en Python dont les lments sont des objets de type int.
  • Pour accder aux lments de T, on utilise son index :

La suite de Collatz¶

Sur le site wikipedia.org, on trouve les informations suivantes :

La suite de Syracuse(ou de Collatz) d'un nombre entier $N>0$ est dfinie par rcurrence, de la manire suivante : $$u_0=N\text{ et pour tout entier naturel }n\text{ : }u_{n+1}=\begin{cases}\dfrac{u_n}{2}\text{ si }u_n\text{ est pair,}\\3u_n+1\text{ si }u_n\text{ est impair.}\end{cases}$$

Conjecture de Syracuse¶

La conjecture de Syracuse affirme que pour tout $N$, il existe un indice $n$ tel que $u_n = 1$.

L'observation graphique de la suite pour $N = 15$ et pour $N = 127$ montre que la suite peut s'lever assez haut avant de retomber. Les graphiques font penser la chute chaotique d'un grlon ou bien la trajectoire d'une feuille emporte par le vent. De cette observation est n tout un vocabulaire imag : on parlera du vol de la suite.

Pour allez plus loin¶

On dfinit alors :

  • le temps de vol en altitude : c'est le plus petit indice $n$ tel que $u_{n+1} \leq u_0$.
    Il est de 10 pour la suite de Syracuse $15$ et de $23$ pour la suite de Syracuse $127$ ;
  • l'altitude maximale : c'est la valeur maximale de la suite.
    Elle est de $160$ pour la suite de Syracuse $15$ et de $4\,372$ pour la suite de Syracuse $127$.
